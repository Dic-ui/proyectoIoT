# 🌎 Sistema IoT - Monitoreo de Calidad del Aire

Proyecto desarrollado para la UNAD.

## 🚀 Descripción
Dashboard web que muestra en tiempo real:
- PM2.5
- Temperatura
- Humedad

## 🧠 Tecnologías usadas
- HTML
- CSS
- JavaScript
- Firebase Realtime Database

## 🔗 Funcionamiento
Los datos son enviados desde un sistema IoT (ESP32 + sensor) hacia Firebase, y luego consumidos por este dashboard.

## 📊 Objetivo
Visualizar la calidad del aire en zonas vulnerables de Bogotá.
